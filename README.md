# 🧭 峰哥工作陪伴 — VS Code Extension

[English README](./README_EN.md)

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

基于“现实观察 + 执行推进”风格的 VS Code 陪伴插件。

## 功能

- 聊天面板：`峰哥观察室`
- 番茄钟专注与休整
- 错误巡查与简报
- 保存彩蛋、闲置提醒
- 今日统计（番茄 / 对话 / 保存）
- 人设策略可配置：`fengge.personaMode` 与 `fengge.safeMode`

## 命令

- `峰哥：开启观察模式`
- `峰哥：开始番茄钟专注`
- `峰哥：结束番茄钟`
- `峰哥：给我打气`
- `峰哥：巡查我的代码`
- `峰哥：今日收工`
- `峰哥：现实独白`
- `峰哥：现场笔记`
- `峰哥：取个绰号`

## 从源码运行

```bash
npm install
npm run compile
```

按 F5 启动 Extension Development Host。
