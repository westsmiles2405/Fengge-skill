# 🧭 Fengge Companion — VS Code Extension

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

A VS Code companion extension inspired by a **realistic observer + execution-driven** style.

## Features

- Chat panel: `Fengge Observation Room`
- Pomodoro focus & break loop
- Diagnostic-based code check summary
- Save-triggered easter eggs and idle reminders
- Daily counters (pomodoros / messages / saves)
- **Persona strategy controls**: `personaMode` (`neutral` / `standard` / `intense`) and `safeMode`

## Commands

- `Fengge: Open Observation Mode`
- `Fengge: Start Pomodoro Focus`
- `Fengge: Stop Pomodoro`
- `Fengge: Encourage Me`
- `Fengge: Review My Code`
- `Fengge: Finish Today`
- `Fengge: Monologue`
- `Fengge: Field Note`
- `Fengge: Give Me a Nickname`

## Run from source

```bash
npm install
npm run compile
```

Press **F5** to launch the Extension Development Host.
